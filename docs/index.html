特徴

シンプルで確実なキャプチャ

🎯

ウィンドウ直接キャプチャ

Quartz API でウィンドウを直接キャプチャ。他のウィンドウに隠れていてもOK。

🔄

自動ページ送り

AppleScript でページ送りを自動化。待ち時間は自由に調整可能。

🧠

ページ変更検出

ピクセル比較でページが実際に変わったことを確認してからキャプチャ。

📄

途中でPDF化

キャプチャ途中でも取得済みページからPDFを生成。全ページ待つ必要なし。

👁

リアルタイムプレビュー

キャプチャ中のページをリアルタイムで確認。進捗バーで残りページも把握。

🔐

権限が持続する署名

コード署名により、アップデート後もプライバシー権限の再設定が不要。

インストール

ソースからビルド

リポジトリをクローン

git clone https://github.com/norinux/kindle-capture.git

依存関係をインストール

python3 -m venv .venv && .venv/bin/pip install PyQt6 Pillow pyobjc-framework-Quartz pyobjc-framework-Cocoa py2app

署名用証明書を作成（初回のみ）

アップデート時の権限再設定を不要にします

./build.sh --setup

ビルド & インストール

./build.sh && cp -R "dist/Kindle Capture.app" /Applications/

必要な権限

  • 画面収録 — Kindleウィンドウのスクリーンショット撮影に必要
  • アクセシビリティ — キーボード操作によるページ送りに必要
